几何系统

        
表情辨识优先于动效。静止画面必须先成立，动画只在此基础上做 1–3 帧的开合或位移。

      

      

        

          
一个母体，四种明确姿态。

          
IDLE、THINKING、WORKING、PERMISSION 与 COMPACTING 共用原始直眼，让 Mochi 始终保持安静稳定；DONE、ERROR 和 SLEEPING 分别使用内夹、交叉与水平闭合表达结果。
